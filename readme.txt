Automatically completes WooCommerce orders that contain only virtual and/or downloadable products.

== Description ==

This lightweight plugin automatically sets WooCommerce orders to "Completed" when they contain only virtual and/or downloadable products.

No configuration needed — simply install and activate.

== Features ==

* Automatically completes orders with only virtual or downloadable products.
* Reduces manual effort for store owners selling digital goods.
* Clean and simple code with no performance impact.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That’s it! The plugin runs silently in the background.

== Frequently Asked Questions ==

= Does this complete orders that contain physical products? =
No. Only orders with 100% virtual/downloadable products are automatically completed.

= Can I customize the logic? =
Yes, you are free to modify the plugin code as needed.
